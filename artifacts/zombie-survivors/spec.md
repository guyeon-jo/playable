# Zombie Survivors — spec

## 개요

브라우저에서 즉시 실행되는 5분 완결 오토슈터 게임. 플레이어는 이동만 담당하고, 캐릭터가 자동으로 공격한다. 레벨업마다 스킬을 선택해 점점 강해지고, 5분을 생존하면 클리어된다. 첫 플레이 완료 후 검사 캐릭터가 해금된다.

---

## 범위

### 포함

- 캐릭터 선택 화면 (첫 방문: 총잡이만 활성, 이후: 검사 추가 활성)
- 게임 플레이 화면: 3×3 화면 크기의 맵, 카메라가 플레이어 추적
- HUD: 남은 시간 타이머, HP 바, 경험치/레벨 바, 현재 장착 스킬 목록
- 캐릭터 2종: 총잡이(원거리), 검사(근접)
- 적 3종: 일반 좀비, 달리는 좀비, 탱커
- 스킬 풀: 총잡이 5종, 검사 5종 (캐릭터별 별도)
- 레벨업 시 게임 일시정지 + 3개 스킬 카드 선택
- 적 처치 시 경험치 구슬 드롭, 플레이어가 닿으면 획득
- 좀비 스폰 강도: 30초마다 증가
- 클리어 화면 (5분 생존 시)
- 게임오버 화면 (HP 0 시)
- 검사 캐릭터 해금: 첫 플레이 완료(클리어 또는 게임오버) 후 영구 저장

### 제외

- 3종 이상 캐릭터 — 밸런싱 부채
- 무기 진화/조합 시스템 — 복잡도 과다
- 모바일/터치 조작 — 별도 feature
- 사운드/BGM — post-MVP
- 리더보드·점수 저장 — post-MVP
- 멀티플레이 — 범위 초과

---

## 시나리오

### 1. 첫 방문 — 캐릭터 선택 화면

- **Given** 게임에 처음 접속했거나 검사가 아직 해금되지 않은 상태
- **When** 게임 페이지를 연다
- **Then** 캐릭터 선택 화면이 표시된다. 총잡이 카드는 선택 가능하고, 검사 카드는 잠금 상태(자물쇠 아이콘 + "첫 플레이 후 해금" 텍스트)로 표시된다

성공 기준:
- [ ] 총잡이 카드를 클릭하면 게임이 시작된다
- [ ] 검사 카드를 클릭해도 반응이 없다(또는 "아직 잠겨 있습니다" 안내)

---

### 2. 총잡이로 게임 시작

- **Given** 캐릭터 선택 화면에서 총잡이가 선택된 상태
- **When** 총잡이 카드를 클릭한다
- **Then** 게임이 시작된다. HUD에 "5:00" 타이머, HP 바(가득 참), 레벨 1 / EXP 0 바, 빈 스킬 목록이 표시된다. 플레이어 캐릭터(총잡이)가 맵 중앙에 나타나고 카메라가 해당 위치를 보여준다

성공 기준:
- [ ] 게임 시작 직후 타이머에 "5:00"이 표시된다
- [ ] HP 바가 가득 찬 상태로 표시된다
- [ ] 레벨 "1"과 EXP 바가 표시된다

---

### 3. 이동 및 카메라 추적

- **Given** 게임이 진행 중이고 플레이어가 맵 중앙 부근에 있다
- **When** WASD 또는 방향키를 눌러 이동한다
- **Then** 캐릭터가 해당 방향으로 움직이고, 카메라가 캐릭터를 화면 중앙에 유지하며 따라간다. 맵 외곽 벽에 가까워지면 카메라가 벽 너머로 넘어가지 않는다

성공 기준:
- [ ] 캐릭터가 맵 외곽 벽에 닿으면 그 방향으로 더 이동하지 않는다
- [ ] 이동 중 카메라가 캐릭터를 화면 중앙에 유지한다

---

### 4. 자동 공격 — 총잡이

- **Given** 게임 시작 직후 총잡이 캐릭터로 플레이 중
- **When** 적 좀비가 공격 사거리 안에 들어온다
- **Then** 총잡이가 자동으로 가장 가까운 좀비 방향으로 투사체를 발사한다. 투사체가 좀비에 맞으면 좀비의 HP가 감소하는 시각적 피드백(피격 이펙트)이 나타난다

성공 기준:
- [ ] 공격 범위 안에 적이 있으면 투사체가 자동 발사된다
- [ ] 투사체가 적에 닿으면 적 HP가 줄어드는 피드백이 보인다

---

### 5. 자동 공격 — 검사

- **Given** 검사 캐릭터로 플레이 중
- **When** 적 좀비가 근접 공격 범위 안에 들어온다
- **Then** 검사가 주변 범위를 자동으로 공격한다(투사체 없이 주변 영역 피해). 범위 안의 모든 좀비가 피격된다

성공 기준:
- [ ] 근접 범위 안의 적 전체에 피해가 들어간다
- [ ] 총잡이와 달리 투사체가 발사되지 않는다

---

### 6. 경험치 수집 및 레벨업

- **Given** 게임 진행 중, EXP 바가 가득 차지 않은 상태
- **When** 적을 처치하면 경험치 구슬이 드롭되고, 캐릭터가 구슬 위를 밟는다
- **Then** EXP 바가 증가한다. 바가 가득 차면 레벨 숫자가 1 올라가고, 게임이 일시정지되며 스킬 카드 3장이 화면 중앙에 표시된다

성공 기준:
- [ ] 경험치 구슬에 닿으면 EXP 바가 증가한다
- [ ] EXP 바가 가득 차면 레벨 숫자가 올라간다
- [ ] 레벨업 시 게임이 멈추고 스킬 카드 3장이 표시된다
- [ ] 카드에는 스킬 이름과 설명이 표시된다

---

### 7. 레벨업 스킬 선택

- **Given** 레벨업 스킬 카드 3장이 표시된 일시정지 상태
- **When** 카드 1장을 클릭한다
- **Then** 게임이 재개된다. 선택한 스킬이 HUD의 스킬 목록에 추가된다. 이미 보유한 스킬이 다시 선택되면 해당 스킬이 강화(레벨 표시 증가)된다

성공 기준:
- [ ] 카드 클릭 후 게임이 즉시 재개된다
- [ ] 선택한 스킬 이름이 HUD 스킬 목록에 나타난다
- [ ] 같은 스킬을 두 번 선택하면 스킬 옆에 레벨 표시(예: Lv.2)가 나타난다

---

### 8. 스폰 강도 증가

- **Given** 게임이 시작된 상태
- **When** 30초가 경과한다
- **Then** 화면에 스폰되는 좀비의 수 또는 속도가 눈에 띄게 증가한다. 이 패턴이 매 30초마다 반복된다

성공 기준:
- [ ] 30초 시점 이후 적 밀도가 시작 시점보다 증가한다
- [ ] 4분 30초 이후(마지막 단계)가 0초 직후보다 명확히 더 많은 적이 등장한다

---

### 9. 피해 수신 및 HP 감소

- **Given** 게임이 진행 중
- **When** 적 좀비가 플레이어에게 접촉한다
- **Then** 플레이어 HP 바가 감소한다. 피격 시 캐릭터에 시각적 피드백(깜빡임 등)이 나타난다. 동일한 적에게 연속 피격되지 않도록 무적 시간이 존재한다

성공 기준:
- [ ] 적 접촉 시 HP 바가 줄어드는 것이 보인다
- [ ] 피격 직후 캐릭터가 짧게 깜빡인다

---

### 10. 게임오버

- **Given** 플레이어 HP가 남아 있는 상태
- **When** 적의 공격으로 HP가 0이 된다
- **Then** 게임이 멈추고 게임오버 화면이 표시된다. 화면에는 생존 시간, 처치한 좀비 수, 달성한 레벨이 표시된다. "다시 시작" 버튼과 "캐릭터 선택으로" 버튼이 표시된다

성공 기준:
- [ ] HP 0이 되면 게임이 즉시 멈추고 게임오버 화면이 보인다
- [ ] 생존 시간, 처치 수, 레벨 3가지 정보가 표시된다
- [ ] "다시 시작" 클릭 시 같은 캐릭터로 새 게임이 시작된다
- [ ] "캐릭터 선택으로" 클릭 시 캐릭터 선택 화면으로 이동한다

---

### 11. 검사 해금 (첫 플레이 완료 후)

- **Given** 최초 플레이로 게임오버 또는 클리어가 발생한 직후
- **When** 결과 화면(게임오버 또는 클리어)이 표시된다
- **Then** 결과 화면에 "검사 해금!" 알림이 표시된다. 이후 캐릭터 선택 화면으로 돌아가면 검사 카드가 선택 가능한 상태로 바뀐다. 브라우저를 새로고침해도 해금 상태가 유지된다

성공 기준:
- [ ] 첫 플레이 결과 화면에 검사 해금 알림이 표시된다
- [ ] 캐릭터 선택 화면에서 검사 카드의 잠금 UI가 사라진다
- [ ] 브라우저 새로고침 후에도 검사 카드가 선택 가능하다

---

### 12. 클리어

- **Given** 게임이 진행 중
- **When** 타이머가 0:00에 도달한다
- **Then** 게임이 멈추고 클리어 화면이 표시된다. 화면에는 처치한 좀비 수, 달성한 레벨, "생존 성공!" 텍스트가 표시된다. "다시 시작" 버튼과 "캐릭터 선택으로" 버튼이 표시된다

성공 기준:
- [ ] 타이머가 0:00이 되면 클리어 화면이 표시된다
- [ ] 클리어 화면에 처치 수와 달성 레벨이 표시된다

---

### 13. 검사로 플레이

- **Given** 검사가 해금된 상태이고 캐릭터 선택 화면에서 검사를 선택함
- **When** 게임이 시작된다
- **Then** 레벨업 시 스킬 카드가 검사 전용 5종(검 회전, 돌진 베기, 충격파, 흡혈 베기, 방패 강타) 중에서 제시된다. 총잡이 전용 스킬(권총 등)은 등장하지 않는다

성공 기준:
- [ ] 검사로 플레이 시 레벨업 카드에 검사 전용 스킬만 표시된다
- [ ] 총잡이 스킬이 검사 레벨업 카드에 등장하지 않는다

---

## 불변 규칙

**게임 일관성**
- 레벨업 선택 화면이 열려 있는 동안 적·투사체·타이머가 모두 멈춘다 — 어떤 시나리오에서도 선택 중 피해를 받거나 타이머가 줄어들지 않는다
- 캐릭터 해금 상태는 게임 결과(클리어/게임오버)가 발생한 즉시 저장되어, 이후 어떤 경로(새로고침, 재방문)로 접근해도 유지된다

**성능**
- 적이 100마리 이상 화면에 존재해도 게임 루프가 눈에 띄게 끊기지 않는다
